# Sales Assistant product

Example second product. To use:

1. Prepare OpenClaw state (agents, prompts, workspace) for a “sales assistant” flow.
2. Build state tarball: copy `../default/build-state-tgz.sh` here and run `./build-state-tgz.sh /path/to/.openclaw`, or create a `state-template/` like in `products/default/`.
3. Install with: `./install.sh --product sales-assistant`.
